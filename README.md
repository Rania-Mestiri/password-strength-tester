# Password Strength Tester

## Project Overview
This project is a web-based Password Strength Tester developed to evaluate and improve password security in accordance with the OWASP (Open Web Application Security Project) guidelines. 

## Objectives
- Evaluate password strength (Weak / Medium / Strong)
- Enforce password security rules
- Provide feedback to improve passwords

## Features
- Username and password input
- Password analysis (length, complexity, entropy)
- Strength scoring system
- Real-time feedback

## System Design
The system includes:
- User Interface
- Input Validation Module
- Password Analysis Module
- Strength Evaluation Engine
- Feedback Generator

## Technologies Used
- HTML, CSS, JavaScript
- Visual Studio Code
